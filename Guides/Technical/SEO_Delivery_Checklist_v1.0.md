# SEO Delivery Checklist v1.0

v1.0 — SEO checklist for every client site before delivery

A step-by-step checklist that runs through before marking any client site as "delivered." Organized by category.

## TECHNICAL SEO

- [ ] Page loads in under 3 seconds (test with PageSpeed Insights)
- [ ] Mobile-responsive (test with Google Mobile-Friendly Test)
- [ ] SSL certificate active (https://)
- [ ] Sitemap.xml created and submitted
- [ ] Robots.txt configured
- [ ] All images compressed and have alt tags
- [ ] No broken links (test with browser extension)
- [ ] Proper URL structure (clean, keyword-friendly)
- [ ] 404 page configured

## ON-PAGE SEO

- [ ] Title tag includes business name + city + main service (under 60 chars)
- [ ] Meta description includes CTA + city + service (under 160 chars)
- [ ] H1 tag on every page (only one H1)
- [ ] H2/H3 hierarchy is logical
- [ ] NAP (Name, Address, Phone) consistent everywhere
- [ ] Schema.org LocalBusiness markup (JSON-LD)
- [ ] Click-to-call on mobile phone numbers
- [ ] Google Maps embed
- [ ] Business hours visible
- [ ] At least 300 words of unique content per page

## LOCAL SEO SETUP

- [ ] Google Business Profile claimed/created
- [ ] GBP: correct category, hours, phone, photos
- [ ] GBP: website URL linked
- [ ] Bing Places listing submitted
- [ ] Apple Maps listing submitted
- [ ] 3 directory listings (Yelp, Yellow Pages, industry-specific)

## CONVERSION OPTIMIZATION

- [ ] Clear CTA above the fold
- [ ] Phone number in header (sticky on mobile)
- [ ] Contact form working (test submission)
- [ ] Social proof (reviews/testimonials visible)
- [ ] Trust signals (licenses, insurance, years in business)
