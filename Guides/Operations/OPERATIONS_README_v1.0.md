## RED FLAGS (Stop Everything & Fix)

**Email Issues:**
- Open rate <25% → Possible deliverability problem (check SPF/DKIM records)
- Bounce rate >5% → List quality issue (validate emails before sending)
- Spam complaints → PAUSE all campaigns immediately

**Sales Issues:**
- Reply rate 0% for 3 days → Subject lines are bad; A/B test new ones
- No calls scheduled → Sales script needs work or list quality is low

**Client Issues:**
- Site down → Check Netlify, check DNS, redeploy
- Angry client → Call within 2 hours, fix the issue, call back next day
- Overdue payment 30+ days → Call the client

**Capacity Issues:**
- Can't build sites fast enough → Use Claude to generate HTML or hire VA
- Too many calls → Price is too low; raise it 20%
- Too many client requests → Hire support VA
