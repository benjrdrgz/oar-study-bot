## RED FLAGS (Stop & Fix Immediately)

**Email Deliverability:**
- Open rate <25% → Check SPF/DKIM/DMARC records or bounce rate
- Bounce rate >5% → List quality issue (validate emails before sending)
- Spam complaints >0 → PAUSE campaigns immediately

**Sales Metrics:**
- Reply rate 0% for 3 days → Subject lines are wrong; A/B test new ones
- No calls scheduled in a week → Sales call script needs work or lead quality is low

**Client Issues:**
- Site is down → Check Netlify status → Check DNS records → Redeploy
- Client unhappy → Reply within 2 hours, fix the issue, follow up next day
- Payment overdue 30+ days → Call the client, not email

**Capacity Issues:**
- Can't keep up with site builds → Build faster (use Claude to generate HTML) or hire VA
- Too many calls scheduled → Price is too low; raise it 20-30%
- Too many clients needing help → Hire support VA for 5-10 hours/week
