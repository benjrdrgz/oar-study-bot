# Referral Program v1.0

v1.0 — Client referral program: structure, terms, and marketing copy

## PROGRAM STRUCTURE

Existing client refers a new business → new client signs up → referrer gets $100 credit on next monthly bill

- No limit on referrals
- Both parties benefit: referrer saves money, new client gets a proven service

## TERMS

- Referral must result in a signed agreement and setup fee payment
- Credit applied to the billing cycle following the new client's first payment
- Referral credit cannot exceed the monthly fee (no negative balance)
- Referral tracked via simple code or mention during sign-up
