QUICK START (3 STEPS)
═════════════════════════════════════════════════════════════════════════════

Step 1: Navigate to scripts folder
  $ cd ~/Website\ Agency/Scripts

Step 2: Run fill_contract.py with client details
  $ python3 fill_contract.py \
      --agency "Your Agency Name" \
      --client "Client Name" \
      --business "Business Name" \
      --date "2026-03-25" \
      --state "Wisconsin" \
      --billing-day "15"

Step 3: Print, customize, sign, and send to client
  → Output file created in /Website Agency/Templates/Contracts/
  → Open in Word/Docs
  → Print for signature
  → Both parties sign and date

RECOMMENDED WORKFLOW
═══════════════════════════════════════════════════════════════════════════════

1. CLIENT SAYS YES → Customer decides on service tier
2. RUN SCRIPT → python3 fill_contract.py (customize with their info)
3. REVIEW CONTRACT → Open in Word/Docs, verify all placeholders filled
4. PRINT & SIGN → Print 2 copies, both parties sign
5. PAYMENT → Setup fee must be received before work begins
6. FILE & TRACK → Save signed copy, track 3-month term end date
7. DELIVER → Build their website, deploy, launch

═════════════════════════════════════════════════════════════════════════════════

Start Date + ongoing → Monthly billing on [BILLING_DAY]
Days past due + 30 days → Service suspension date (if unpaid)
